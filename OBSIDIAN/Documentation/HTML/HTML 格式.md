## 文本格式化标签

```html
<b>      定义粗体文本。</b> 
<big>    定义大号字。 </big> 
<em>     定义着重文字。</em>
<i>      定义斜体字。 </i>  
<small>  定义小号字。 </small>
<strong> 定义加重语气。</strong> 
<sub>    定义下标字。 </sub>
<sup>    定义上标字。 </sup>
<ins>    定义插入字。 </ins>
<del>	 定义删除字。</del>

```
